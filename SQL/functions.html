MIN() - returns the smallest value of the selected column e.g., SELECT MIN(column) FROM table WHERE condition; SELECT MIN(Price) AS SmallestPrice FROM Products; MAX() - returns the largest value of the selected column COUNT() - returns the number of rows that matches a specified criteria e.g., SELECT COUNT(column) FROM table WHERE condition; AVG() - returns the average value of a numeric column SUM() - returns the total sum of a numeric column
