About SQL - Structured Query Language - used for storing, manipulating, and retrieving data in databases - is American National Standards Institute (ANSI) standard - is International Organization for Standardization (ISO) standard Different SQLs - MySQL, SQL Server, MS Access, Oracle, Sybase, Informix, Postgres - But major commands are similar: SELECT, UPDATED, DELETE, INSERT, WHERE RDBMS - Relational Database Management System - basis for SQL - data in RDBMS is stored in database objects called "tables" Table Structure - Table: collection of related data entries and consists columns and roww - Each table has a name (e.g., "Customers" or "Orders") - Field: smaller entities tables are broken into - Record: same as row; each entry (e.g., one record for each customer) - Column: vertical entity (e.g., CustomerID, CustomerName) To user SQL, you need - RDBMS database program (e.g., MS Access, MySQL) - server-side scripting language (e.g., PHP, ASP) - HTML/CSS for styling Resources: - w3schools.com
