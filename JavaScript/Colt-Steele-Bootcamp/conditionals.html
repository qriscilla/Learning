One of the most important concepts in programming! 3 JS conditional keywords: if - takes a single condition and runs code block if it is true else if - follows an if-statement and is a secondary condition else - "otherwise"; run code block if none of the above conditions were met Club Bouncer Example If you are younger than 18 you cannot enter the club If you are between 18 and 21 you can enter but cannot drink Otherwise you can enter and drink Exercise If age is negative print an error message If age is 21 print "Happy 21st birthday!"
